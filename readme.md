### Introduction
OpenHacks is an enhancement component for the foobar2000 player.

Requirements:
- Windows 10 1607+
- foobar2000 v1.5+
- foobar2000 v2.0+ x86/x64 

### Features
- Hide DUI menu bar
- Hide DUI status bar
- Draggable and resizable borderless main window
- ...

### Build
Visual Studio 2022
